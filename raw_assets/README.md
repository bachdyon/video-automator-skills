# raw_assets/

Drop image, video, or audio files anywhere in this folder and the watcher will index them automatically.

The watcher recursively scans the entire tree, so any subfolder organisation is fine. Suggested layout (purely a convention):

- `images/` for photos and stills
- `videos/` for clips
- `audio/` for narration, music, and sound effects

Files in this folder are intentionally git-ignored (they are usually large and binary). Only this README and `.gitkeep` are tracked.
